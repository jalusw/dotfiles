# dotfiles

This repository contains my dotfiles configuration. I use [GNU
Stow](https://www.gnu.org/software/stow/) to manage them and personally  i use
ubuntu based distro.


## Getting Started

clone the repository

```bash
git clone https://github.com/jalusw dotfiles
```

Run `setup.sh`
```
chmod +x ./setup.sh
./setup.sh
```
